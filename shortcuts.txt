Shift + Shift - Поиск везде
Ctrl + Shift + Enter - Интеллектуальное автозавершение
Ctrl + Alt + L - Форматирование кода
Alt + Enter - Варианты исправления кода
Ctrl + P - Параметры функции
Ctrl + D - Дублирование текущей строки или блока
Ctrl + Y - Удаление текущей строки или блока
Alt + Shift + Up/Down - Перемещение строки
Shift + Enter - Новая строка
Alt + Left Click (Колесико) - Множественный курсор
Alt + J - Выделить следующее вхождение
Ctrl + / - Однострочный комментарий
Ctrl + Shift + / - Многострочный комментарий
Ctrl + Shift + J - Объединение выделенных строк в одну строку
Ctr + F/R - Поиск и замена
Ctrl + Shift + U - Изменение регистра
Ctrl + W - Расширить выделение 
Ctrl + Shift + W - Уменьшить выделение
Ctrl + Alt + S - Настройки
